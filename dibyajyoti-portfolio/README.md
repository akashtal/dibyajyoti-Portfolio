## Contact API (optional SMTP)

Set environment variables to enable email sending:

- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_USER`
- `SMTP_PASS`
- `CONTACT_TO` (optional; defaults to `SMTP_USER`)

Without these, messages are logged to the server console in development.
